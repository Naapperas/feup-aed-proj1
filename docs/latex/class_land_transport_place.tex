\doxysubsection{Detailed Description}
Represents a land transport (subway, train, bus) near an airport An object of \mbox{\hyperlink{class_land_transport_place}{Land\+Transport\+Place}} is relative to only one \mbox{\hyperlink{class_airport}{Airport}} (if there are two airports close to each other, and both are close to a local train station, each one keeps their own \mbox{\hyperlink{class_land_transport_place}{Land\+Transport\+Place}} for that train station, mainly because of the different distances to each airport)
